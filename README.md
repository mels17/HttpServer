<h1> Http Server </h1>

To run the cobspec tests: 
1. Go to the terminal.
2. Navigate to the cobspec folder.
3. There is a shell script - **cobspec.sh**:
```
#!/bin/sh
/Library/Java/JavaVirtualMachines/jdk1.8.0_161.jdk/Contents/Home/bin/java -jar fitnesse.jar -p 9090
```
4. Run this command on the terminal:
```
./cobspec.sh
```
